jQuery Notification Plugin

Notify is a jQuery plugin that makes it easy to create alert, success, error, warning, information, messages as an alternative the standard alert dialog. Each modal notification is added to a queue.

The notifications can be positioned at the top - topLeft - topRight - center - bottom - bottomLeft - bottomRight

There are lots of other options in the API to customise the text, speed and much more.

It also has various callbacks for opening and closing the notifications.

You can also easily create your own template or layout

This plugin require jQuery, jQuery UI core and jQuery UI widget

Inline notification

$('#not1').notify();

  • closeWith

    Type
    Array
    Default
    ['click']

    Event to close the notification

    Code examples

    Initialize a notification with the closeWidth option specified.
    $.notify({closeWidth:['button']});
    Get or set the closeWidth, after init.
    This method is not yet implemented

  • debug

    Type
    Boolean
    Default
    false

    Active (true) or unactive (false) de debug output.
    You must use firebug.

    Code examples

    Initialize a notification with the debug option specified.
    $.notify({debug:true});
    Get or set the debug, after init.
    $.notify("option", "debug", true);

  • force

    Type
    Boolean
    Default
    false

    Put the notification on the top of the queue

    Code examples

    Initialize a notification with the force option specified.
    $.notify({force:true});

  • layout

    Type
    String
    Default
    'base'

    The layout name you want to choose for your notification

    You can easily create your own layout (see Theming tab)

    Code examples

    Initialize a notification with the layout option specified.
    $.notify({layout:'topRight'});

    Type

  • beforeShow

    Type
    notifybeforeshow

    This event is triggered just before the notification is displayed

    Code examples

    Supply a callback function to handle the beforeShow event as an init option.
    $.notify({ beforeShow: function(event, ui) { ... } });
    Bind to the beforeShow event by type: notifybeforeshow.
    $(".selector").bind("notifybeforeshow", function(event, ui) { ... });

  • inWaitingList

    Signature
    .notify( "inWaitingList" ) : Boolean

    Return true if the notification is in the waiting list. This method is available only with modal notifications.

  • open

    Signature
    .notify( "open" )

    Show the notification. Only possible with show: false.

  • close

    Signature
    .notify( "close" )

    Close the notification.
